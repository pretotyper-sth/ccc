# 챗 퍼널에서 바뀐 것 (한국어·일본어 공통)

페이지에서 **눈으로 보이는 변경**만 정리한 문서예요.

---

## 1. 참여 시기 선택지 하나 추가

**어디서:** 챗 12단계 중 **「참여는 언제쯤으로 희망하시나요?」** 단계

**기존 4개**
- 1주일 이내
- 1개월 이내
- 2~3개월 이내
- 미정 (정보 받으면 그때 결정할게요)

**추가된 1개 (5번째)**
- **한국 올 일정이 정해지면 연락해 주세요** (한국어)
- **韓国に行く予定が決まったら知らせてほしい** (일본어)

→ “지금은 일정 없는데, 한국 갈 때 알려달라” 수요용.

---

## 2. 제출 완료 화면에 안내 문구 하나 추가

**어디서:** **참여 신청하기** 누르고 제출이 성공한 뒤 나오는 **등록 완료** 화면

**기존**
- 제목: 등록 완료!
- 문구: 세션 매칭 시 이메일로 연락드릴게요.
- 버튼: 처음으로 / 最初に戻る

**추가된 문구 (아래에 한 줄 더)**
- **한국어:**  
  「한국 오시는 일정이 잡히면 이 메일로 알려주시면, 그 시기에 맞춰 세션 안내해 드릴게요.」
- **일본어:**  
  「韓国へ行く日程が決まったら、このメールに返信でお知らせいただければ、その時期に合わせてセッションをご案内します。」

→ “일정 나중에 잡히면 이 메일로 알려주면 된다”는 걸 명시.

---

## 3. 화면에는 안 보이는 변경 (데이터·분석용)

- **Formspree 전송:**  
  위에서 5번째 옵션(한국 올 일정 알림)을 선택하고 제출하면, 폼에 **`_remind_when_travel: "1"`** 이 함께 전송됨.  
  → 나중에 “한국 올 때 알림” 리스트만 따로 쓰려면 Formspree/시트에서 이 필드로 필터하면 됨.

- **GA4 이벤트:**  
  제출 클릭/성공/실패 이벤트에 **`form_source`** (chat_kr / chat_jp) 가 붙음.  
  → 일본 전환 0건일 때 “진입 부족 vs 제출 실패” 구분하려고 넣은 것.

---

## 요약

| 구분 | 변경 내용 |
|------|-----------|
| **참여 시기** | 선택지 5번째 추가: 「한국 올 일정이 정해지면 연락해 주세요」 |
| **완료 화면** | 문구 한 줄 추가: 일정 잡히면 이 메일로 알려주면 된다는 안내 |
| **데이터** | 5번째 선택 시 `_remind_when_travel: 1` 전송, GA4에 `form_source` 포함 |

한국어·일본어 페이지 **둘 다** 위 내용이 동일하게 적용돼 있어요.
